== Description ==

Integrated Google ReCaptcha for WordPress. Protect the login, register, lostpassword and comment forms. Support Woocommerce, Ultimate Member and more popular forms.
<a href="https://developers.google.com/recaptcha">**Google reCAPTCHA**</a> is a free service that protects your site from spam and abuse. It uses advanced risk analysis techniques to tell humans and bots apart.

= Features =

Go to <a href="https://www.google.com/recaptcha/admin/create">Google reCAPTCHA</a> to get the **Site key** and **Secret key**.

= reCAPTCHA Type: =
* reCAPTCHA **v3**
* reCAPTCHA **v2 Checkbox**
* reCAPTCHA **v2 Invisible**

= Supported Form List =
* Wordpress default login form
* Wordpress default register form
* Wordpress default lostpassword form
* Wordpress default comment form

* <a href="https://wordpress.org/plugins/woocommerce/" target="_blank" >**Woocommerce**</a> login form
* <a href="https://wordpress.org/plugins/woocommerce/" target="_blank" >**Woocommerce**</a> register form
* <a href="https://wordpress.org/plugins/woocommerce/" target="_blank" >**Woocommerce**</a> lostpassword form
* <a href="https://wordpress.org/plugins/woocommerce/" target="_blank" >**Woocommerce**</a> checkout Billing form
* Add a shortcode **[toms_woo_register_form]** for **woocommerce register form** on any page you want.

* <a href="https://wordpress.org/plugins/ultimate-member/" target="_blank" >**Ultimate Member**</a> login form
* <a href="https://wordpress.org/plugins/ultimate-member/" target="_blank" >**Ultimate Member**</a> register form
* <a href="https://wordpress.org/plugins/ultimate-member/" target="_blank" >**Ultimate Member**</a> lostpassword form

* <a href="https://wordpress.org/plugins/contact-form-block/" target="_blank" >**Contact Form Block**</a> Contact Form Block
* more support forms comming soon...


= Option settings =
* Verify API : **Google.com**/**Recaptcha.net** **---Notice:---** Some country can not use Google verify API, that means Google verify API will not work, even using vpn. If google.com not work try use Recaptcha.net
* reCAPTCHA v2 (Checkbox)  Theme: **Light**/**Dark**
* reCAPTCHA v2 (Invisible) Badge: **Bottom Right**/**Bottom Left**

== Frequently Asked Questions ==

= Why the captcha not show in the login form ? =

Go to **Admin Panel** -> **TomS WP** -> **TomS reCAPTCHA** -> check the **Login form** and click **Save Change**.
